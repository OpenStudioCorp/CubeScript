First, let's start with the basics. CubeScript is a scripting language based around the C style family of languages.

What does that mean? Well, it means that CubeScript is a language that is based around the C programming language. If you've ever programmed in C, C++, C#, Java, or JavaScript, you'll find that CubeScript is very similar to those languages.

Let's start with the basics. In CubeScript, there are two types of values: numbers and strings. Numbers are just that, numbers. Strings are a sequence of characters. For example, "Hello, world!" is a string. 123 is a number.

Let's start with numbers. Numbers are pretty simple. You can add, subtract, multiply, and divide them. You can also use parentheses to group operations. For example, 1 + 2 * 3 is 7, but (1 + 2) * 3 is 9.
