CubeScript How to

Defining a function

Functions are defined using the Struct keyword. 


Struct main (string{} args)
{
    
}

Functions are a way to group code together and execute it when needed. Functions can be called from other functions or from the console.

Calling a function

When calling a function, just call it by name and pass any arguments it needs in the parentheses. 


Struct speak(string text)
{
    print("Hello {text}");
}

speak("World");

Returning a value

Functions can return a value using the return keyword. 


Struct add(int a, int b)
{
    return a + b;
}

int result = add(1, 2);
print(result); // 3

Defining arrays

there are 2 different arrays. Static and Dynamic. 


Struct main (string{} args)
{
    // this launguage also supports arrays, which are defined like this

    string{} names = {"John Doe", "Jane Doe", "Bob Doe"}; // string array
    
    // you can also use the var keyword to make the compiler automatically detect the type of the array and set it at runtime
    
    var names = {"John Doe", "Jane Doe", "Bob Doe"}; // string array
    
    //there are 2 different types of arrays, static and dynamic
    
    // static arrays are arrays that have a fixed size and cannot be changed after it has been set
    // you can change the contents of the array but when you try to add or remove more then what the array can store or has from the array, it will throw an error
    // when you run out of the caracters you have in the array, any blank fields will return null
    
    //static arrays are defined like this
    
    string[3] names = {"John Doe", "Jane Doe", "Bob Doe"}; // string array
    
    
    // dynamic arrays are arrays that have a dynamic size and can be changed after it has been set
    // you can change the contents of the array and you can also add or remove an item from the array and blank spots will be removed and not replaced with null
    
    //dynamic arrays are defined like this
    
    string{} names = {"John Doe", "Jane Doe", "Bob Doe"}; // string array
    
    // you can also use the var keyword to make the compiler automatically detect the type of the array and set it at runtime
    
    var  names = {"John Doe", "Jane Doe", "Bob Doe"}; // string array
    
    // notice how the brackets for defining the size of the array are missing, this is because the compiler will automatically detect the type of the array and set it at runtime
    // nice and simple
    
    // you can also use the const keyword to make the array a constant, meaning that it cannot be changed after it has been set, even if it is a var array
    
    // now ive gotten some concern on how dynamic arrays resize on lets say, checking how many users in the instance of a game are online.
    
    // the dynamic array will, if defined to do so, will check the max plyayer count will be what of the world.
    // if the max player count is 10, then the array will be 10, if the max player count is 100, then the array will be 100, and so on and so forth
    // saves people from having to define the array size and then having to change it later on
    }

"For" controll structure

if you want to iterate through something, you can use the For keyword. 


Struct main (string{} args)
{
    string{} names = {"John Doe", "Jane Doe", "Bob Doe"}; // string array

    for(name in names{})
    {
        print(name);
    }
}

For loops are a way to iterate through an array or a string.

"While" and "If" controll structure
