Defining a function

Functions are defined using the struct keyword. 


struct main (string{} args)
{
    
}

Functions are a way to group code together and execute it when needed. Functions can be called from other functions or from the console.

you can also do public struct or private struct if you need it public or private

Calling a function

When calling a function, just call it by name and pass any arguments it needs in the parentheses. 


restring speak(string text)
{
    Print($"Hello {text}");
}

speak("World");

Returning a value

Functions can return a value using the return keyword. 


reint add(int a, int b)
{
    return a + b;
}

int result = add(1, 2);
Print(result); // 3

"for" controll structure

if you want to iterate through something, you can use the For keyword. 


public struct main (string[] args)
{
    string{} names = {"John Doe", "Jane Doe", "Bob Doe"}; // string array

    for(name > names)
    {
        Print(name);
    }
}

For loops are a way to iterate through an array or a string.
